# Conferra

Conferra is a real-time video conferencing platform built with WebRTC, Socket.io, Node.js, Express.js, and React. The application enables seamless virtual communication through video calls, meeting rooms, live chat, and screen sharing features.

## Features

- Real-time video and audio communication
- Meeting room creation and joining
- Live chat messaging
- Screen sharing support
- Peer-to-peer communication using WebRTC
- Real-time updates with Socket.io

## Tech Stack

### Frontend
- React
- Vite
- CSS

### Backend
- Node.js
- Express.js
- Socket.io

### Real-Time Communication
- WebRTC
